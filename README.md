# CS 210 Project Three: Corner Grocer

## Project Summary
This project is a grocery tracking program written in C++. It reads a file of items purchased during the day and counts how often each item appears. The program allows the user to search for a specific item, display all item frequencies, and view a histogram. This helps analyze which items are purchased most often.

## What I Did Well
I created a clear and user-friendly menu system. I also successfully read data from a file and displayed it in different formats, including a histogram.

## Where I Could Improve
I could improve my program by adding better input validation and error handling. This would make the program more reliable and secure.

## Most Challenging Part
The most challenging part was reading from the file and counting how often each item appeared. I overcame this by breaking the problem into smaller steps and using maps.

## Transferable Skills
This project helped me develop skills in file handling, data structures, and organizing code. These skills can be used in many future programming projects.

## Maintainability, Readability, and Adaptability
I used clear variable names, comments, and structured code to make the program easy to read and maintain. The program is adaptable because it reads from an input file, so it can work with different data.
